# Contributing

This is a personal project for educational purposes. 

## No Pull Requests

We do not accept pull requests. If you want to make changes, please fork the repository and modify it yourself.

## Issues

Issues are open for bug reports, but there is no guarantee of response or fixes.

## Forks

Feel free to fork and modify this project for your own use. That's the intended way to use this project.
